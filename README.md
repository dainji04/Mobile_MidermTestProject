# Login Application

Ứng dụng Android đơn giản với chức năng đăng nhập sử dụng Material Design Components.

## Công nghệ sử dụng

- **Ngôn ngữ**: Java/Kotlin
- **Build tool**: Gradle
- **UI Framework**: Material Design Components
- **IDE**: Android Studio Koala Feature Drop | 2024.1.2 Patch 1

## Tính năng

- Giao diện đăng nhập với Material Design
- Xác thực username và password
- Hiển thị thông báo lỗi khi nhập thiếu thông tin
